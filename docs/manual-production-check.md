# ReefChart Lite 本番手動確認

対象URL: <https://coral-db.vercel.app>

本番アカウントでログインし、確認前の入力値を控えてから実施する。
テスト後は必要に応じて元の値へ戻す。

## 1. プロフィール保存

1. `/lite/profile` を開く。
2. 次の任意項目を入力する。
   - 水槽の幅（cm）
   - 水槽の奥行（cm）
   - 水槽の高さ（cm）
   - 実水量（L）
   - pH
   - 人工海水
   - 照明
   - ウェーブポンプ
   - 濾過方式
3. 保存を実行し、エラーが表示されないことを確認する。
4. ページを再読み込みする。
5. 保存したすべての値が残っていることを確認する。

## 2. ショップカルテ

1. `/lite/shop-card` を開く。
2. 「飼育環境」セクションが表示されることを確認する。
3. 水槽図が横スクロールなしで表示されることを確認する。
4. 水槽本体をタップし、次の情報が表示されることを確認する。
   - 水槽サイズ（幅 × 奥行 × 高さ cm）
   - 実水量（L）
   - 濾過方式
5. 照明、ウェーブポンプ、人工海水、pHをそれぞれタップし、保存した内容が表示されることを確認する。

## 3. 未入力時の表示

1. `/lite/profile` で寸法、実水量、pH、人工海水、照明、ウェーブポンプ、濾過方式をすべて空にして保存する。
2. 再読み込み後も空のままであることを確認する。
3. `/lite/shop-card` で「飼育環境」セクション全体が表示されないことを確認する。
4. 「未入力」が項目ごとに大量表示されないことを確認する。
5. 必要に応じて確認前の値を再入力して保存する。

## 4. 匿名共有

1. 任意項目を入力済みの状態で共有リンクを発行する。
2. URLを控え、ログインしていない別ブラウザまたはプライベートウィンドウで開く。
3. 水槽図と「飼育環境」が表示されることを確認する。
4. 水槽本体と各パーツをタップし、保存した内容が表示されることを確認する。
5. 個人情報や内部的なデータ名が表示されないことを確認する。
6. ログイン中の画面で共有を停止する。
7. 匿名側のページを再読み込みし、水槽データを閲覧できないことを確認する。

## 5. Lite主5項目

次の主5項目について、記録画面で入力・保存し、ショップカルテに表示されることを確認する。

- KH（炭酸塩硬度）
- 水温
- 塩分濃度
- 硝酸塩（NO3）
- リン酸塩（PO4）

pHが主5項目の通常測定カードに混ざらず、ショップ相談用の任意情報として扱われていることも確認する。
