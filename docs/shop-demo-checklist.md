# ReefChart Lite ショップデモ確認項目

## デモの開き方

1. ReefChart Lite のトップを開く
2. 「ナガレハナ不調デモを見る」を押す
3. スマートフォンを店員へ渡し、説明せずに画面を見てもらう

## ショップに聞く質問

- この画面だけで水槽の状況は分かりますか？
- 足りない情報はありますか？
- いらない情報はありますか？
- 初心者のお客さんに使ってもらえそうですか？
- このアプリを店頭で勧めたいと思いますか？
- ナガレハナ不調の相談として、次に何を確認しますか？

## 観察すること

- 5秒以内に水槽サイズと問題の傾向を把握できたか
- 水質、添加剤、写真のどこを最初に見たか
- 画面を拡大したり、説明を求めたりした箇所はどこか
- 購入案内ではなく相談のためのカルテとして信頼できるか
