# ReefChart Project Context

## ReefChart概要

ReefChartは、リーフアクアリウムの記録を整理し、飼育者とショップが同じ事実を見ながら相談できるようにするプロダクトである。数値を保存するだけでなく、測定日時、水換え、添加剤、写真などを、判断の根拠が追える形で扱う。

## Liteの思想

ReefChart Liteは、初心者や高齢の利用者でも続けやすい無料の記録ツールである。主項目はKH（炭酸塩硬度）、水温、塩分濃度、硝酸塩（NO3）、リン酸塩（PO4）の5つに限定し、部分入力を認める。

Liteで最も重要な画面はショップカルテである。約5秒で読めること、診断を装わないこと、不要な購入を促さないことを守る。pH、人工海水、照明、ウェーブポンプ、濾過方式は任意の補足情報であり、未入力でもLiteの利用を妨げない。

## Proの思想

ReefChart Proは、中級者・上級者が自分で詳細な分析を行うための有料プロダクトである。Ca、Mg、追加水質項目、長期傾向、予測、相関、イベント分析などはProが担当する。

LiteとProは別のデータ領域と操作体験を持つ。Proの機能をLiteへ段階的に混ぜる設計は採用しない。

## Honest Engine

Honest Engineは、ReefChart全体に適用する判断原則である。

- 実測値、手入力、計算値、推定値を区別する
- 欠損や古い測定を隠さない
- 相関を因果関係として断定しない
- 根拠が弱いときは再測定またはショップ相談へ案内する
- 単発の通常範囲外値から購入や添加を勧めない
- 複数の危険値があるときは商品提案よりショップ相談を優先する

Honest Engineは診断AIの名称ではなく、表示、判定、提案、共有に共通する設計ルールである。

## 現在実装済み

- Lite水槽プロフィールと主5項目の部分測定
- 水換え、使用中添加剤、水槽写真の記録
- 主5項目の信号表示と測定鮮度表示
- アプリ内ショップカルテ
- 期限付き共有リンク、共有停止、ショップフィードバック
- Honest Engineに沿った確認・相談ルーティング
- ショップレビュー用デモ水槽
- ReefChart Pro v1の独立した基盤
- Web/PWAとCapacitor Androidレビュー用構成

任意の飼育環境項目は実装中であり、migration、入力、保存、ショップ表示、テストを同じ変更単位で完成させる。

## 今後のロードマップ

1. Liteの任意飼育環境項目を完成し、実機でショップカルテを確認する
2. Supabaseのmigration、RLS、共有RPCを本番で検証する
3. Liteのモバイル入力とショップ相談フローを改善する
4. Web/PWAの安定後、Androidのローカル保存と同期を段階的に実装する
5. Proの分析機能を、Liteの単純さを変えずに独立して拡張する

ロードマップは提供時期の保証ではない。各段階でユーザー、ショップ、開発者の三者にとっての価値と安全性を再評価する。

## ブランド名

- ブランド: **ReefChart**
- 無料版: **ReefChart Lite**
- 高度分析版: **ReefChart Pro**
- 共通原則: **Honest Engine**

`Coral DB`、`Aqua Reef Log`、`ReefCard` は旧名称としてもユーザー向けUIへ復活させない。

## 開発ルール

1. Liteの主5項目を増やさず、追加情報は明確に任意とする
2. Ca、Mg、ICP、消費速度、高度分析をLiteへ入れない
3. 未入力項目を大量の「未入力」表示に変えない
4. DB内部名、enum、UUIDをユーザー向けUIへ出さない
5. DB、型・正規化、UI、保存、表示、テストを同じ変更で整合させる
6. RLSを前提とし、共有データは有効な期限付きトークン経由に限定する
7. 実装済み、開発中、予定をドキュメント上で混同しない
8. 変更後に `git diff --check`、テスト、buildを実行する
9. 保存データを破壊する変更にはmigrationと移行方針を用意する
10. 不明なことを確定事項として表示しない
